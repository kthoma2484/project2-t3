Welcome to Da-Ta-Baby - the baby name finder!

Just pick an option for how you would like to search for a baby name. Each option allows you to filter your search based on the type of search you select. The random name is a luck of the draw search. The search by origin will request a last name from you, search that last name, and find suggest first and/or middle names that have the same country origin. And finally, the search by common U.S. names will suggest names based on popular names recorded in U.S. records since 1880! We hope you enjoy our application selections! Get ready to find your baby's perfect name!

Select one of the search items to begin!

Instructions for 'random name search':


  • First, select how many names you wish the full name to include - e.g. a first, one middle, and/or two middle names.

  • Next, select the gender of the name you wish to search for. If gender is irrelevant, select 'either'.

  • Then, type the last name you want included with the full name in the input box shown.

  • Finally, press the 'search' button to begin your search!

Instructions for 'Inspired by origin search':


  • First, type the name last for which you wish to find the origin.



  • Next, select the gender of the name you wish to search for. If gender is irrelevant, select 'either'.

  • Then, select the how many names you wish the full name to include - e.g. a first, one middle, and/or two middle names. You will need to enter the letter you wish each name to begin with for the search.

  • Finally, press the 'search' button to begin your search!

Instructions for 'common U.S. names search':


  • First, select the gender of the name you wish to search for. If gender is irrelevant, select 'either'.

  • Then, select the decade you want to search for name popularity during that time period.

  • Finally, press the 'search' button to begin your search!
